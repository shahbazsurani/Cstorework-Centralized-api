Reminders Module

Comprehensive reminder system with items, categories, schedules, and more

Authentication Required: All reminders endpoints require valid authentication token.

Dashboard & Search

GET /reminders/dashboard
Get dashboard overview with statistics and recent items.
GET /reminders/search
Search through reminder items.

Items Management

GET /reminders/items
Retrieve a list of all reminder items.
POST /reminders/items
Create a new reminder item.
GET /reminders/items/{item}
Get details of a specific reminder item.
PUT PATCH /reminders/items/{item}
Update an existing reminder item.
DELETE /reminders/items/{item}
Delete a reminder item.
POST /reminders/items/{item}/complete
Mark a reminder item as complete.
POST /reminders/items/{item}/reopen
Reopen a completed reminder item.

Subtasks

GET /reminders/items/{item}/subtasks
List all subtasks for a specific item.
POST /reminders/items/{item}/subtasks
Create a new subtask for an item.
GET /reminders/items/{item}/subtasks/{subtask}
Get details of a specific subtask.
PUT PATCH /reminders/items/{item}/subtasks/{subtask}
Update a subtask.
DELETE /reminders/items/{item}/subtasks/{subtask}
Delete a subtask.
